// prep_your_rig

Show up ready to dump. A little prep = a lot more points.

$ pack

  • Laptop + charger (Linux/WSL/macOS — you’ll want a real shell)
  • USB-TTL / FT232 serial adapter + jumper wires
  • A cheap SOIC-8 clip + CH341A flasher (for SPI flash dumps)
  • Multimeter (find UART, don’t fry the board)
  • The cursed device itself. And a backup cursed device.
  • Power strip / USB hub — outlets are always scarce

$ apt install

  • binwalk
  • flashrom
  • minicom / picocom / screen
  • sigrok + pulseview (logic analyzer)
  • ghidra / cutter
  • firmware-mod-kit, sasquatch, jefferson

[ rules_of_engagement ] Anything on the table is fair game — that’s the whole point. Drop a device on the pile and it’s open season: bringing it is your consent to watch it get dumped, and you rack up points cracking everyone else’s, too. Off-limits: the venue’s network, people’s personal phones/laptops, and anything you didn’t bring to the party. Findings get disclosed responsibly. Hack the things on the table; don’t be a felon.
